Game Rules
1. User will be presented with a question along with 4 options to choose from.
2. The score on the top left corner increment by 5 for every correct answer.
3. If the answer is incorrect you will not be awarded any points.
4. User will have 30 seconds to answer all the questions.
5. If the user answers all question or the user runs out of time, they will be presented with submit form.
6. User will also have access to all check the other scores by clicking the View Highscores button.
7. Start playing the game by clicking the 'Start Quiz' button.
